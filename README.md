# Intelligent Candidate Discovery

## Overview
This project ranks candidates for a given job description using a hybrid AI-based ranking pipeline.

## Features
- Career history analysis
- Skill matching
- Keyword relevance
- Behavioral signal scoring
- Explainable candidate reasoning
- Final ranked Top-100 candidate recommendations

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Project Structure

candidate_ranking.ipynb
submission.xlsx
requirements.txt
README.md

## Output

The final output is a ranked list of the Top 100 candidates with:
- Candidate ID
- Rank
- Score
- Recruiter-style reasoning
